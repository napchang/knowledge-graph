# Knowledge Graph

Obsidian-style knowledge graph for AI intelligence visualization.

## Live Site

**https://napchang.github.io/knowledge-graph**

## Data Source

Data is deployed from the private pipeline repository via GitHub Actions.

## Tech Stack

- D3.js v7 (force-directed layout)
- Vite
- GitHub Pages

## Development

```bash
npm install
npm run dev
npm run build
```
